# Ethiqia Frontend (Next.js)
Frontend minimal (Next.js App Router + Tailwind) para Ethiqia.

## Variables
Copia `.env.example` a `.env.local` y ajusta:
NEXT_PUBLIC_API_BASE=https://ethiqia-backend.onrender.com

## Scripts
- npm run dev
- npm run build
- npm start

## Despliegue en Render
- Build command: npm install && npm run build
- Start command: npm start
- Select Node 18+
